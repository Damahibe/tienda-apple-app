# 🍺 Mercado Aura POS

Sistema de Punto de Venta (POS) moderno para bares, cafeterías y tiendas, desarrollado con HTML, CSS y JavaScript.

## ✨ Descripción

Mercado Aura POS es una aplicación web diseñada para agilizar la gestión de ventas, pedidos e inventario en pequeños negocios como bares, cafés y tiendas.

La interfaz está optimizada para diferentes tipos de usuarios mediante perfiles específicos, permitiendo una experiencia intuitiva tanto para vendedores como para administradores y clientes.

## 🚀 Funcionalidades

### 🛒 Gestión de Ventas

* Carrito de compras dinámico.
* Cálculo automático de subtotal, servicio y total.
* Cobro rápido de pedidos.
* Limpieza de carrito con un clic.

### 🍻 Catálogo de Productos

* Visualización de productos en tarjetas.
* Búsqueda instantánea.
* Filtrado por categorías.
* Contador de productos disponibles.

### 👥 Múltiples Roles

* Vendedor.
* Administrador.
* Cliente.

Cada perfil muestra información y herramientas adaptadas a sus necesidades.

### 📦 Gestión de Inventario

* Control visual de inventario.
* Detección de productos con stock bajo.
* Panel administrativo dedicado.

### 📊 Dashboard Administrativo

* Ventas del día.
* Margen bruto.
* Alertas de inventario.
* Exportación de reportes.

### 💳 Métodos de Pago

* Tarjeta.
* Efectivo.
* QR.

### 🍽️ Servicio para Mesas

* Registro de cliente o número de mesa.
* Cambio entre modos de servicio.
* Flujo optimizado para restaurantes y cafeterías.

## 🛠️ Tecnologías Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ▶️ Ejecución Local

1. Clonar el repositorio:

```bash

```

2. Abrir la carpeta del proyecto.

3. Ejecutar utilizando Live Server en Visual Studio Code o abrir directamente `index.html`.
